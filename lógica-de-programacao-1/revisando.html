Em HTML os comandos vêm dentro de tags que são representadas dessa forma: < e >


A maioria das tags têm abertura e fechamento, como a tag que nos permite programar em JavaScript: <script> e </script>, mas algumas tags são autocontidas e só precisam ser iniciadas, é o caso da tag que faz a quebra de texto (ou pula linha), <br>.


Para criar um título, usamos o par de tags <h1> e </h1>

.

E para fazer uma quebra no parágrafo utilizamos qual tag? Isso mesmo! <br>.


Para criar um pop-up de alerta em JavaScript utilizamos a função "alert ()", mas lembre que ela deve estar dentro da tag <script>
